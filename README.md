## tmux
Source tmux .config:
`tmux source ~/.dotfiles/.config/tmux/.tmux.conf`

In your root directory run:
`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`

open tmux and run `prefix + I`
